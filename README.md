# TroubleinTerroristTown
There's trouble amuck!

I created a website for my friends not familiar with how to play the game Trouble in Terrorist Town on Garry's Mod (on Steam) to have the resources available to win in the harsh environment that is TTT. I created a home page describing the basic idea of TTT and why I made this webpage in the first place. I have an items page to at least give my friends a couple of items to use from the shop in game. There are a lot of items in that shop, so it can be confusing on which ones to use and not use. I have a strategy page that describes how to actually play the game. The game does not make a lot of sense at first. It seems like a traditional shooter, until you realize there's much more you can do than just shoot people. The grid design was a bit confusing at first, but eventually I got it to where I wanted, which is the current items page. The color scheme is alright for now. I decided on maroon and white because those seemed like easy colors for a starting place. I can always change the color scheme later.
